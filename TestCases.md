# Тест-кейси
## JUnit

## 1. Тест-кейс: Додавання нової книги вручну

### Що тестується
Перевірити, що метод ``` addBook() ``` коректно додає книгу до бібліотеки користувача

### Передумови
* Користувач авторизований
* Бібліотека порожня

### Опис

**1. Викликати** ``` addBook() ``` **з даними книги:**
* Назва: "The Hobbit"
* Автор: "J.R.R. Tolkien"
* ISBN: "9824163355125"
* Кількість сторінок: 304
* Статус: "to read"

**2. Отримати список усіх книг користувача**

### Очікуваний результат
* Книга з назвою "The Hobbit" присутня у бібліотеці.
* Розмір бібліотеки = 1
* Усі поля книги відповідають введеним

### Фактичний результат
- [x] Книга була успішно додана
- [x] Розмір бібліотеки = 1
- [x] Дані збігаються

## 2. Тест-кейс: Зміна статусу книги

### Що тестується
Перевірити, що метод ``` updateBookStatus() ``` змінює статус книги правильно

### Передумови
* У бібліотеці є книга "The Hobbit", статус “to read”.

### Опис

**1. Викликати** ``` updateBookStatus("The Hobbit", "finished") ```

**2. Отримати оновлену книгу зі списку**

### Очікуваний результат
* Статус книги змінено на "finished"

### Фактичний результат
- [x] Статус книги змінено успішно

## 3. Тест-кейс: Пошук книги за автором

### Що тестується
Перевірити, що метод ``` findhBooksByAuthor() ``` повертає список книг певного автора

**Передумови**
У бібліотеці є книги:
* "The Hobbit" - J.R.R. Tolkien
* "The Fellowship of the Ring" - J.R.R. Tolkien
* "The Way of Kings" - Brandon Sanderson
* "Children of Time" - Adrian Tchaikovsky

### Опис

**1. Викликати** ``` findhBooksByAuthor("J.R.R. Tolkien") ```

### Очікуваний результат
* Повернуто список із 2 книг ("The Hobbit" і "The Fellowship of the Ring")

### Фактичний результат
- [x] Список містить 2 книги автора J.R.R. Tolkien
